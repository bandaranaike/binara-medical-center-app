# Tasks

This folder contains both raw task intake and normalized task tracking.

## Entry Point

- Add raw tasks in your own language to `inbox/raw-tasks.md`

## Tracking Files

- `index.md`: global task list with status
- `planned/`: ready tasks
- `in-progress/`: active tasks
- `blocked/`: paused tasks
- `done/`: completed tasks
- `cancelled/`: cancelled or replaced tasks

## Rule

Every implementation should have a matching normalized task file.
