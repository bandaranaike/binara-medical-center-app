# Raw Tasks Inbox

Write tasks here in any language.

## How To Use

1. Add a new entry at the top.
2. Use your own language and natural wording.
3. One request per entry is preferred.
4. Include screenshots, links, file paths, or examples if useful.

## Entry Template

